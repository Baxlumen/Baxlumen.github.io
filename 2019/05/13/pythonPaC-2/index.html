##1.通过urllib.request.Request()函数中headers参数修改

所以在headers位置添加参数(一个dict)

import urllib.request
import random

url = ‘https://whatsmyip.com/'
iplist = [‘106.12.147.81:8080’,’112.50.245.92:8080’]

#代理，调用Request的ProxyHandler方法
proxy_support = urllib.request.ProxyHandler({‘https’:random.choice(iplist)})

#这里要区分url是http还是https，相应的proxyHandler的参数也要修改成对应的http还是https

opener = urllib.request.build_opener(proxy_support)
opener.addheaders = [(‘User-Agent’,’Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/69.0.3497.100 Safari/537.36’)]
urllib.request.install_opener(opener)

response = urllib.request.urlopen(url)

html = response.read().decode(‘utf-8’)

print(html)
`

##注：
这里有本机IP和网络IP的区别，本机IP是在局域网之中本机的IP地址，而网络IP是运营商分配的网络地址，在局域网通信中，使用的是本机的IP，例如魔兽，暗黑，红警等的局域网对战；但在网络中IP地址是运营商给的，百度搜索ip就可以看到
